## 查看面板
![](./images/02.png)

> 个人面板默认展示最近一周的，专注学习时长，如果需要查询更长时间的时长，需要在个人设置中进行如下配置：

```json
{
  "studyArea.times":{
    "starTime":"2022-07-00 00:00:00",
    "endTime":"2022-07-00 00:00:00"
  }
}
```
